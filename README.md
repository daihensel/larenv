# Ambiente de Desenvolvimento Laravel com Docker

O ambiente contem as seguinte configuraçoes

* PHP + Apache  = larenv_php_apache
* MariaDB 	= larenv_db
* PhpMyAdmin    = larenv_phpmyadmin

# Network

Por padrao o ambiente utiliza a seguinte rede 10.10.10.0/16 larenv_network

* 10.10.10.2 = PHP + Apache  = larenv_php_apache
* 10.10.10.3 = MariaDB 	     = larenv_db
* 10.10.10.4 = PhpMyAdmin    = larenv_phpmyadmin
